ELite Tech
=============
It's a store management system

#Introduction
===============
ELite Tech is a store management system designed to help you calculate profits, track the number of devices, and generate general statistics for your inventory.

#Installation
================
Python3 - Postgresql
================
python3 -m venv env
On Linux and Mac :
   source env/bin/activate
On Windows :
   .\env\Scripts\activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

#Usage
=========
Go to add devices and fill in the entries and that's it.

#Features
============
Device Management: Add, delete, and edit devices.
Statistics: View sold, rented, and available devices in graphical format.
Profit Calculation: Automatically calculate profits from rented and sold devices.
Device Categories: Organize devices into categories such as Mobile, Laptop, Gaming, etc.
